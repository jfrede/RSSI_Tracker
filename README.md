RSSI_Tracker
============

RSSI Tracker for FPV use

Used Hardware:   Arduino Lenonardo
                 LCD4884 Shield For Arduino
                 32CH 5.8G Diversity Receiver 7" D702 Plus Monitor
                 2x 9g Servos
                 Hobbyking Pan/Tilt Mechanic
                 Piezo Beeper
                 5V Voltage Regulator
                 2 x AD8541 OpAmp
                 4 x 10k Resistor
                 3 x 47k Restistor
                 1 x Poti 50K Linear
                 1 x Switch
